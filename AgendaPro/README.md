# AgendaPro

Sistema de gestión de citas y agenda empresarial desarrollado en C# Windows Forms y SQL Server.

## Descripción

AgendaPro permite gestionar citas para servicios profesionales o técnicos, controlando clientes, profesionales, horarios y estados de atención.

## Funcionalidades MVP

- Inicio de sesión por roles
- CRUD de clientes
- CRUD de profesionales
- CRUD de servicios
- Gestión de citas
- Validación de horarios disponibles
- Reportes de citas
- Búsquedas por cliente, profesional y estado

## Tecnologías utilizadas
